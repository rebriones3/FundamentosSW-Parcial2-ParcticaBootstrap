Preguntas frecuentes

¿Qué es HTML?

HTML es el lenguaje que permite crear la estructura de una página web.

¿Para qué sirve una etiqueta?

Una etiqueta indica al navegador cómo debe interpretar una parte del contenido.

Texto técnico en HTML

Para guardar un archivo se puede usar el atajo: Ctrl + S.

Una etiqueta se puede mostrar como texto usando entidades: <h1>Título principal</h1>
